# Keymonk Modern DualSwipe v0.3 Safe Keyboard Prototype

This version is intentionally conservative.

Changes in v0.3:
- Single-letter taps remain the only text-entry behavior.
- Swipes do NOT write into the text field.
- Two-finger swipe paths are shown only in the keyboard debug bar.
- Backspace, space, and enter are handled as real keyboard actions.
- This build is meant to verify the keyboard can behave normally before adding prediction.
